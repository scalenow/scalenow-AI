# scalenowAI Platform

## Overview
scalenowAI is an AI-orchestrated enterprise platform integrating project management, document management, knowledge systems, and learning platforms into a unified experience.

## Core Capabilities
- Natural language interaction via OpenWebUI
- AI-driven orchestration layer
- Cross-platform search using YaCy
- Unified identity with Keycloak
